It's a two-leg robot that doesn't let itself fall. It uses Nema 17 Stepper Motor and A4988 Stepper Motor Driver with some Arduino programming To Balance this robot and using A Gyro Sensor(MPU6050) which continuously sends the robot orientation data to the controller. Stepper Motor React On the Data That Is Send By Mpu6050, It Moves For-word If Robot Is Falling in front side and Its Move Back-word If Robot Is falling Back Side.
We Can Also Move It In Left-Right, For-ward, and Backward Direction With The Help Of Our SmartphoneThrough Bluetooth that's mean Its is Bluetooth Control Self Balancing Robot.
